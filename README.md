# 조사장닷컴(chosajang.com)
개인 사이트를 직접 기획부터 개발까지 모든 것을 다 해보고 싶어서 시작한 프로젝트입니다.

Front/Back-end 두개의 프로젝트를 진행하고 있으며, github.io 도메인 사용을 위한 웹서비스용 저장소를 포함한 총3개의 저장소를 사용합니다.

> **Front-end 프로젝트** - Vue.js + jQuery + Bootstrap + Awesome Font

## 기획
